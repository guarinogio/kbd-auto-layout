# Changelog

## 1.0.0
- First stable public release
- Stable X11 keyboard layout auto-switching daemon
- Per-device rules with `exact` and `contains` matching
- Default fallback layout support
- User systemd service
- Debian package with man pages
- JSON output for CLI inspection commands
- Zsh completion
- Automated tests for CLI, daemon rule selection, matching, and Debian build flow

## 0.2.0
- Added Debian package build improvements
- Added man pages
- Added JSON output for list and status
- Added `init-config`
- Added `set-poll-interval`
- Added zsh completion
- Added `exact` and `contains` match modes
- Included example config in the package

## 0.1.1
- Added Debian packaging
- Added man pages
- Added JSON output for CLI status and list
- Added `init-config`
- Added `set-poll-interval`
- Added zsh completion
- Added device matching modes: `exact` and `contains`

## 0.1.0
